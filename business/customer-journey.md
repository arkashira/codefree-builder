# customer-journey.md  

## Overview  
**Product:** `codefree-builder` – a no‑code platform that lets founders, side‑hustlers, and indie hackers spin up SaaS, marketplaces, or internal tools in days instead of months.  

| Persona | Core Goal | Primary Pain |
|---------|-----------|--------------|
| **Tech Startup Founder** | Validate a MVP fast | “I need to ship before the runway runs out, but I can’t hire devs.” |
| **Digital Side‑Hustler** | Launch a revenue stream with minimal overhead | “I have ideas but no coding chops; outsourcing is too expensive.” |
| **Indie Hacker** | Build & iterate solo | “Manual coding drags me down; I lose momentum.” |

The journey map below follows the five classic phases: **Aware → Consider → Try → Adopt → Expand**. Each phase lists the *trigger event*, *friction points*, *user emotions*, *opportunities to delight*, and a *key metric* to track success.

---

## 1. Aware  

| Element | Detail |
|---------|--------|
| **Trigger Event** | • Social‑media ad (Twitter/LinkedIn) showcasing “Launch a SaaS in 48 h – No code required.” <br>• Guest post on Indie Hackers titled “How I built my first product without writing a line of code.” |
| **Friction Points** | • Skepticism about “no‑code” quality.<br>• Over‑saturation of generic builder ads (signal‑to‑noise low). |
| **User Emotions** | Curiosity → Mild doubt → Hopeful intrigue. |
| **Opportunities to Delight** | • Use **real‑world case studies** (e.g., a $10K/mo marketplace built in 3 days). <br>• Offer a **short 30‑second explainer video** that shows a live build. <br>• Include a **trust badge** (e.g., “Used by 5,000+ founders”). |
| **Metric** | **Impressions → Click‑through Rate (CTR)**. Target: **≥2 % CTR** on ad placements; **≥5 %** on editorial mentions. |

---

## 2. Consider  

| Element | Detail |
|---------|--------|
| **Trigger Event** | • Visitor lands on the landing page after clicking the ad or article. |
| **Friction Points** | • Unclear pricing tiers.<br>• Uncertainty about feature limits (e.g., “Can I integrate Stripe?”).<br>• Fear of vendor lock‑in. |
| **User Emotions** | Analytical → Overwhelm → Decision anxiety. |
| **Opportunities to Delight** | • **Interactive pricing calculator** that shows monthly cost based on projected usage. <br>• **Feature comparison matrix** vs. manual coding & top competitors. <br>• **Live chat with a product specialist** (30‑second response SLA). <br>• **Free “Starter Pack”** of 5 pre‑built templates (e‑commerce, SaaS dashboard, blog, etc.). |
| **Metric** | **Consideration Conversion Rate** – % of visitors who request a demo, start a free trial, or download a template. Target: **≥12 %**. |

---

## 3. Try  

| Element | Detail |
|---------|--------|
| **Trigger Event** | • User clicks “Start Free Trial” (14‑day, no credit‑card). |
| **Friction Points** | • Onboarding length (too many steps).<br>• Lack of guidance on “first project”.<br>• Performance hiccups on first build (slow rendering). |
| **User Emotions** | Excitement → Frustration (if stuck) → Satisfaction (if success). |
| **Opportunities to Delight** | • **Guided “Wizard” onboarding** that builds a sample app in <5 min. <br>• **In‑app “coach” chatbot** powered by the same LLM, offering instant help. <br>• **Community Slack channel** with a “#first‑build” channel; push a welcome message with a “quick‑win” checklist. <br>• **Performance guarantee**: if the first build exceeds 10 seconds, we credit 1 extra trial day. |
| **Metric** | **Trial Activation Rate** – % of sign‑ups that complete the first guided build. Target: **≥70 %**. |
| **Secondary Metric** | **Time‑to‑First‑Publish** – average minutes from sign‑up to live URL. Goal: **≤15 min**. |

---

## 4. Adopt  

| Element | Detail |
|---------|--------|
| **Trigger Event** | • User publishes a live MVP (or upgrades to a paid plan). |
| **Friction Points** | • Billing confusion (monthly vs. annual).<br>• Limited integrations (e.g., Zapier, HubSpot).<br>• Scaling concerns (traffic spikes). |
| **User Emotions** | Confidence → Pride → Cautious optimism. |
| **Opportunities to Delight** | • **One‑click upgrade** with a clear “What you get” overlay. <br>• **Integration marketplace** with pre‑built connectors; auto‑populate API keys. <br>• **Performance dashboard** with real‑time traffic & cost alerts. <br>• **Customer success check‑in** (email + 15‑min call) within 7 days of upgrade. |
| **Metric** | **Paid Conversion Rate** – % of trial users who become paying customers. Target: **≥25 %** (industry benchmark for no‑code SaaS). |
| **Secondary Metric** | **30‑day Net Retention (NRR)** – aim for **≥110 %** (upsell + cross‑sell). |

---

## 5. Expand  

| Element | Detail |
|---------|--------|
| **Trigger Event** | • Existing customer adds new projects, purchases add‑ons, or refers peers. |
| **Friction Points** | • Pricing opacity for add‑ons.<br>• Lack of advanced customization (e.g., custom JS). |
| **User Emotions** | Loyalty → Empowerment → Advocacy. |
| **Opportunities to Delight** | • **Tiered “Growth Pack”** (advanced analytics, white‑labeling) with a free 30‑day trial. <br>• **Referral program**: “Invite a founder, get 2 months free.” <br>• **Annual “Founder Summit”** (virtual) with product roadmap sneak‑peeks. <br>• **Co‑creation lab**: invite power users to beta‑test new modules; reward with credits. |
| **Metric** | **Expansion Revenue per User (ERPU)** – average additional ARR from existing customers. Target: **≥$250 / yr**. |
| **Secondary Metric** | **Referral Conversion Rate** – % of referred sign‑ups that become paying users. Target: **≥15 %**. |

---

### Quick Reference Table  

| Phase | Trigger | Key Friction | Core Emotion | Delight Lever | Success Metric |
|-------|---------|--------------|--------------|---------------|----------------|
| **Aware** | Ad / article click | Trust skepticism | Curiosity → Doubt | Real‑world case study video | CTR ≥ 2 % |
| **Consider** | Landing page visit | Pricing/feature opacity | Analysis → Overwhelm | Pricing calculator + live chat | Consideration conv. ≥ 12 % |
| **Try** | Free‑trial start | Onboarding length, performance | Excitement → Frustration | Guided wizard + coach bot | Trial activation ≥ 70 % |
| **Adopt** | First live MVP / upgrade | Billing, scaling | Confidence → Pride | One‑click upgrade + dashboard | Paid conversion ≥ 25 % |
| **Expand** | New project / referral | Add‑on pricing, customization | Loyalty → Advocacy | Growth pack trial + referral credits | ERPU ≥ $250 yr, Referral conv. ≥ 15 % |

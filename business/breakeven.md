# breakeven.md

## Unit Economics & Break-even Analysis for Codefree-Builder

### Cost per Active User
- **Compute Costs**: $5/user/month
- **Storage Costs**: $2/user/month
- **Bandwidth Costs**: $1/user/month  
**Total Cost per Active User**: **$8/user/month**

### Pricing Tiers
| Tier Name       | Price ($/mo) | Features                                           |
|------------------|--------------|----------------------------------------------------|
| **Starter**      | $15          | Basic templates, 1 project, email support          |
| **Professional** | $30          | Advanced templates, 5 projects, priority support   |
| **Enterprise**   | $60          | Custom templates, unlimited projects, dedicated support |

### Customer Acquisition Cost (CAC) Range
- Estimated CAC: **$50 - $100** per user

### Lifetime Value (LTV) Estimate
- Average Revenue per User (ARPU) for different tiers:
  - Starter: $15/month
  - Professional: $30/month
  - Enterprise: $60/month
- Average user lifespan: 24 months
- LTV Calculation:
  - Starter: $15 * 24 = **$360**
  - Professional: $30 * 24 = **$720**
  - Enterprise: $60 * 24 = **$1440**
  
**Average LTV**: Assuming a mix of users, estimated LTV = **$600**

### Break-even Users Count
- Break-even Point = Fixed Costs / (ARPU - Variable Costs)
- Assuming fixed costs (marketing, salaries, etc.) = $10,000/month
- Break-even Users Count = $10,000 / ($600 - $8) = **17 users** (approx.)

### Path to $10K MRR
To achieve $10,000 MRR, we can consider the following tier distribution:

1. **Starter Tier**: 
   - Price: $15
   - Users Needed: $10,000 / $15 = **667 users**

2. **Professional Tier**: 
   - Price: $30
   - Users Needed: $10,000 / $30 = **334 users**

3. **Enterprise Tier**: 
   - Price: $60
   - Users Needed: $10,000 / $60 = **167 users**

### Recommendation
To balance user acquisition and revenue, a mixed approach could be:
- **200 Starter users**: $3,000
- **100 Professional users**: $3,000
- **50 Enterprise users**: $3,000  
**Total MRR**: $9,000 (with additional upsell opportunities)

**Path to $10K MRR**: Focus on acquiring **250 Starter users** or **150 Professional users** or **80 Enterprise users** to reach the target.
